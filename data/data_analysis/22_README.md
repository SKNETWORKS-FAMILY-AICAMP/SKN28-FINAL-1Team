## 1. 이 데이터는 무엇인가

`s3://skn28-cozy/22.사이즈코리아/`는 **한국인 성인 인체치수 조사 공식 데이터**다. 5차(2003~04)부터 8차(2020~24)까지 5개 조사차수(5·6·7·8차)를 Excel 파일로 저장해 두었으며, 각 차수마다 **직접 측정(신체 계측)**과 **3D 스캐닝 측정** 두 가지 유형이 포함되어 있다.

우리 프로젝트와의 관계: 20번 데이터(한국인 전신 형상 및 치수 측정 데이터)가 "1,000명 × 37개 실측치 + 사진 + 3D 메쉬"인 반면, 22번은 **시계열 성격을 갖는 대규모 인구통계 기반 인체치수 데이터**다. 동일 항목(키, 가슴둘레 등)을 20년간 추적 가능.

핵심 자산:

- **5개 차수 시계열** (2003~04년 ~ 2020~24년): 인구통계 기반 대표성 높은 표본. 동일 항목 비교 가능.
- **직접 측정 데이터**: KS K 0051 기반 표준 인체 계측 항목 (신장, 둘레, 길이, 너비 등).

## 5. 가장 중요한 파일

**8차 (2020~24) Excel — (3차년도) 직접측정 시트.**

이유:
- 5개 차수 중 가장 최신
- 1,014명 × 99개 항목 = 약 100,000개 셀
- **20번 데이터(한국인 전신 형상)와 함께 쓰면**: 20번이 2021년 1,000명 심층 데이터(37개 항목 + 사진 + 3D)라면, 22번 8차는 2020~24년 동일 항목의 **대표성 높은 표본**으로 상호 보완
- 시계열 비교: 5차~8차 동일 항목을 추적해 "한국인 체형 변화" 분석 가능

## 6. 샘플 하나를 보는 순서

`ID = 22_0001` (8차 3차년도 측정, 여성, 70세)를 예로 본다.

1. **8차 Excel에서 `(3차년도) 직접측정` 시트를 연다** — `header=0` 옵션 (row 0이 컬럼명).
   - *왜*: row 0에 'ID', '성별', '나이', '001.어깨길이(등 뒤)', '002.키' 등이 위치. row 1부터가 실제 데이터.

2. **4번째 컬럼(컬럼명: '나이')을 확인** — 값 70~84. **나이(70세~84세)Confirmed**.
   - *왜*: 이 표본이 70세 이상 고령자 중심이라는 것을 즉시 확인. 20번 데이터(전 연령)와 비교할 때 Age filter 필수.

3. **5번째 컬럼(컬럼명: '002.키')을 확인** — ID 22_0001의 값 = 1498mm = **키 149.8cm**.
   - *왜*: 키는 신체의 기본 스케일 기준점. 20번 데이터의 '키' 항목과 직접 비교 가능.

4. **성별·나이 분포를 확인** — `여성=573, 남성=441`. 연령대별 분포(70~84세, 고르지 않음) 확인.
   - *왜*: 1,014명 표본이 전체 한국 성인 인구를 얼마나 대표하는지, 특히 고령자 의류 설계를 위해 얼마나 필요한지 가늠.

5. **(1~2차년도) 직접측정 시트로 이동** — 5,097행 × 142열. 더 많은 연령층을 포함.
   - *왜*: 3차 표본(70~84세 고령자)과 비교해 1~2차가 어떤 연령대를 커버하는지 확인. 전체 한국 성인 인구 대표성 파악.

6. **5차~7차 Excel과 구조를 비교** — 5차(15,581행), 7차(6,425행)과 8차(1~2차 5,097행)의 공통 항목 추출.
   - *왜*: 동일 항목(예: 젖가슴둘레)의 시계열 변화 추적. 20년간 한국인 체형이 어떻게 변했는지 정량화.

7. **3D 측정 시트를 본다** — (3차년도) 3D 전신 시트(429행 × 177열)의 ID 포맷 `22_F_0001`과 3D 좌표 항목명 확인.
   - *왜*: 3D 좌표 기반 자동 측정 → 20번의 PLY/OBJ 3D 메쉬와 비교 가능성.

- **왜 그렇게 판단**: 3D 스캐닝 좌표로 계산된 정밀 부분 체형 데이터. 모자·신발·장갑 치수 설계에 활용 가능
- **활용**: 부분 의류(모자, 신발, 장갑) 추천 시스템, 3D 가상 피팅의 부분 매핑

### 8.7 성별 × 연령대 체형 차이

- **무엇**: 8차 3차 표본에서 여성 573명 + 남성 441명의 체형 차이. 고령자 표본(70~84세)이므로 "노년기 체형에서 성별 차이가 어떻게 나타나는지"
- **왜 그렇게 판단**: pandas 직접 분석으로 여성 573 + 남성 441 확인. 1,014명 표본에서 여성/남성 각각의 분포 비교 가능
- **활용**: 성별 기반 체형 추천의 근거 데이터. 노년기 체형에서 성별에 따라 허리둘레/엉덩이둘레 비율이 어떻게 다른지 정량화

## 9. 추천 시스템에 활용하는 방법

### 9.1 체형 시계열 기반 사이즈 추천

5차~8차 동일 항목을 시계열 비교해 "한국인 체형이 커졌다/작아졌다"를 정량화. 이 추세를 추천 시스템에 반영.

### 9.2 고령자 체형 추천

8차 3차(70세 이상) 표본 1,013명의 체형 분포를 학습해 "시니어 패션" 추천에 활용. 20번 데이터와 함께 노년기 체형 프로파일 구성.

### 9.3 대표성 높은 표본으로 cold-start 해결

5,097명(8차 1~2차) + 15,581명(5차) + 6,425명(7차) = **약 27,000명**의 대규모 표본 분포에서 사용자 입력(키, 몸무게, 나이)만으로 가장 가까운 군집 탐색.

### 9.4 3D 체형 좌표 기반 정밀 매칭

3D 자동 측정 데이터(429명 × 177항목)의 3D 좌표를 활용. 20번 데이터의 PLY 3D 메쉬와 좌표 체계 비교 → "3D 좌표 → 의류 치수" 역산 모델.

### 9.5 부분 의류 치수 (모자·신발·장갑)

3D 머리·발·손 스캐닝 데이터 → 모자/신발/장갑 사이즈 추천. 20번 데이터의 실측치와 결합.

### 9.6 장애인가구 Inclusive 추천

휠체어 사용자 체형 데이터를 학습에 포함 → 배달 가구, 특수 설계 의류 추천.

### 9.7 20번 데이터와 결합: 정밀 + 대표성

20번(977명, 37개 항목, 사진+3D) × 22번(5차~8차, 27,000명, 100+ 항목) = "정밀 심층 분석 + 대표성 있는 넓은 분석" 병행.

- **Excel 헤더 레이아웃**: 모든 차수에서 header row는 row 0 (pandas `header=0`). row 0 = 컬럼명, row 1부터 = 데이터. (3차) 시트는 `header=0`만으로 충분.

- **중복 항목**: 동일 항목을 여러 차수에서 다르게 분류할 수 있음. Cross-walking 필요.
- **의류 치수 표준**: 조사 프로토콜은 KS K 0051 기반. 20번 데이터와 동일 표준.
