# AI 패션 추천 앱 — 기능 설계 (마인드맵 전사본)

> 원본: `기능설계.png` (마인드맵 이미지)를 텍스트로 옮긴 문서. 이미지가 화면에서 잘 안 보일 때 이 문서를 참고.

## 1. 개인 맞춤형 룩 추천

### 1.1 오늘의 룩 추천 생성
- 추천 후보 생성(옷장 조합)
- 추천 이유 설명(날씨·TPO·추구미 근거)
- 대안 제시(상/하의 교체, 레이어링)
- 온도/날씨 자동 반영(오늘 기본)
- 위치 권한 요청 및 대체 입력(날씨용)
- 날씨 데이터 조회 및 캐시

### 1.2 추천 룩 피드백
- 좋아요/싫어요 입력 UI
- 피드백 후 재추천

## 2. LLM 대화형 상품 추천

### 2.1 대화 UI
- 대화 입력(텍스트/추천 프롬프트)
- 컨텍스트 유지(옷장/최근 대화 반영)
- 모드 선택(추구미 반영/옷장 기반)
- 새 채팅 생성(세션 생성)

### 2.2 세션 관리
- 대화 기록 목록/상세
- 대화 기록 검색
- 세션 삭제/이름 변경

### 2.3 2D 가상착장
- 룩 구성 카드(아이템별)
- 가상 착장 이미지 생성/표시
- 키/몸무게 입력 및 BMI 계산
- 정면/측면 사진 촬영·업로드
- 사진 업로드 및 서버 전송
- 포즈 추정으로 관절 좌표 추출(서버)
- 픽셀-센티미터 스케일 계산(키 기준)
- 폭/두께 측정(정면/측면)
- 둘레 추정(타원 공식 + BMI 보정)
- 추정 치수 출력 및 사이즈 매칭
- 사진 보관/삭제 정책(90일)

## 3. 옷장 데이터 관리

### 옷장 아이템 등록(사진/수동)
- 사진 업로드 및 기본 정보 자동 추출
- 수동 등록 폼(카테고리/색상/사이즈 등)
- 아이템 정보 수정/삭제

### 옷장 아이템 태그/검색/필터
- 통합 검색(키워드/태그)
- 필터/정렬(카테고리·색상·계절)

### 착용 기록 및 통계
- 아이템별 착용 기록 입력/자동 기록
- 활용 요약(마지막 착용일/저활용 경고)

## 4. 스타일 선호도

### 스타일 온보딩(추구미/기피 요소)
- 추구미/스타일 선택 템플릿
- 기피 요소 설정(색/핏/아이템)

### 피드백 기반 선호도 학습
- 룩 피드백 수집(좋아요/싫어요/이유)
- 피드백 반영 로직(다음 추천 가중치)

### 선호도 설정 관리
- 선호도 설정 화면

## 5. 날씨·TPO 연동

### TPO 입력 및 템플릿
- TPO 빠른 선택(칩/카드)
- TPO 자유 입력 및 제약조건 추출

## 6. 새 아이템 탐색

### 부족 아이템 추천
- 부족 아이템 판정(옷장 대비)
- 추천 아이템 후보 생성

### 아이템 상세 정보 제공
- 아이템 상세(가격대/스타일)

### 외부 커머스 링크 연결
- 외부 링크 열기(딥링크/웹)

### 위시리스트
- 위시리스트 추가/삭제
- 위시리스트 조회/정렬

## 7. 계정·프라이버시

### 가입/로그인
- 이메일 가입/로그인
- 소셜 로그인

### 데이터 권한 동의 관리
- 동의 항목 상세 보기
- 동의 변경 이력 및 적용 범위

### 계정/데이터 삭제
- 탈퇴 요청 및 확인
- 데이터 삭제 범위 선택

## 8. 룩북 저장·재활용

### 저장/북마크(추천 결과)
- 룩 저장 및 태그/메모
- 해시태그 추가/편집

### 룩북 조회/필터
- 룩북 리스트/상세 화면
